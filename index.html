RHODES ISLAND // USER GUIDE

🌐 网站简介

罗德岛通讯终端是一款明日方舟同人角色扮演聊天应用。你可以与罗德岛的干员们进行沉浸式对话，感受每位干员独特的性格与语气。

💬
角色扮演对话

支持 12+ 位罗德岛干员，每位干员拥有独立的人格设定与对话风格，基于 LLM 大语言模型驱动。

🎤
TTS 语音合成

所有角色均可自定义中日双语音色，采用火山引擎豆包 2.0 语音合成 + 声音复刻技术。

🎨
Live2D 动态立绘

支持 Live2D Cubism 模型展示，部分角色拥有专属动态立绘，带来更生动的视觉体验。

🎵
背景音乐系统

内置多首角色主题曲与氛围音乐，支持播放/暂停/切换，营造沉浸式对话氛围。

📎
文件/图片对话

支持上传图片、文档（PDF/Word/Excel/PPT），自动 OCR 提取文字，多模态模型可直接识别图片内容。

📱
响应式设计

完美适配桌面端与移动端，移动端支持滑动切换干员，体验流畅。

🔌 API 接入指南

本应用需要你提供大语言模型 API 才能正常对话。支持所有兼容 OpenAI Chat Completions 格式的 API 服务。

方式一：DeepSeek（推荐，国内可用）

  1. 访问 platform.deepseek.com 注册账号
  2. 进入「API Keys」页面，创建 API Key
  3. 在右侧设置面板中：
    • API 提供商：选择 DeepSeek
    • API Key：粘贴你的 API Key
    • Model：选择 deepseek-v4-pro 或 deepseek-v4-flash
  4. 点击「保存设置」即可开始对话

方式二：OpenAI

  1. 访问 platform.openai.com
  2. 创建 API Key 并充值
  3. 设置面板中选择 OpenAI 提供商，填入 Key，选择模型

方式三：硅基流动（国内中转，无需翻墙）

  1. 访问 siliconflow.cn 注册
  2. 创建 API Key
  3. 提供商选择 硅基流动 (SiliconFlow)
  4. Model 可选用 deepseek-ai/DeepSeek-V3、Qwen/Qwen2.5-72B-Instruct 等

方式四：阿里百炼 / 智谱 / 月之暗面

以上平台均兼容 OpenAI 格式，选择对应提供商或手动输入 Base URL 即可。

自定义接口

  1. 提供商选择「自定义」
  2. 手动输入 API Base URL（如 https://your-api.com/v1）
  3. 填入 API Key 和模型名
  4. 只要能返回 OpenAI 格式的 /chat/completions 响应即可使用

🔊 TTS 语音配置

TTS 采用火山引擎豆包语音合成 2.0，通过自建 CloudBase 代理转发请求。

  1. 注册 火山引擎 账号
  2. 开通「语音合成」服务，获取 API Key
  3. 在 TTS 设置区域填入 API Key
  4. 代理地址已内置自建 CloudBase 代理，留空即可用；也可填入自己的 Cloudflare Worker / 云托管地址覆盖
  5. 去火山引擎「声音复刻」克隆角色声音，获得 S_xxx 格式的音色 ID
  6. 在下方「角色音色映射」区域添加角色并填入对应音色 ID

📖 使用技巧

  • 切换角色：点击右侧干员卡片即可切换对话角色，聊天记录按角色独立保存
  • 清除记忆：点击聊天区域顶部的「CLEAR」按钮可清除当前角色对话记录，开始新话题
  • 语音播放：角色消息旁有播放按钮▶，点击即可 TTS 朗读（仅支持角色）
  • 日语模式：在 TTS 设置区域可切换为日语，消息将自动翻译为日文后合成语音
  • 上传文件：聊天框左侧 📎 按钮支持上传图片和文档，多模态模型可直接理解图片
  • 代码高亮：角色回复中的代码块将自动语法高亮渲染
  • 数学公式：支持 KaTeX 渲染 LaTeX 数学公式（行内 $...$ 和块级 $$...$$）
